# Accurate Clock Crafter 精准时钟合成器

An automation to make Minecraft resourcepacks of minute-level-accurate clock.  
[Demo video](https://youtu.be/B5HugHxZtt0)

[[简体中文介绍]](https://github.com/GrakePch/AccurateClockCrafter/blob/master/README_CN.md)  

## Usage

Rename the clock to following names to get different looks!

- Analog Clock

- Square Analog Clock

- Square Digital Clock

- Giant Analog Clock
